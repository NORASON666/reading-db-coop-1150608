# 智慧閱讀平臺・資料介接協力說明

致 **線上資料庫得標廠商・資料庫維運廠商** 的資料介接配合事項簡報，供 **2026-06-08（民國115年）14:00 協調會** 使用。

🔗 **線上簡報**：https://norason666.github.io/reading-db-coop-1150608/

## 內容

從智慧閱讀系統的角度，說明本年度資料銜接作法與請廠商配合的六件事：

1. 書目欄位灌齊（syncDBBook 24 欄、17 必填）
2. 智慧閱讀視角的關鍵欄位務必提供（含與 Hamibook 欄位對照）
3. 書影規格（檔名＝書籍編號、≤60KB）
4. 書籍編號（bookID）一致性
5. 每日回傳瀏覽紀錄（getDBlog）
6. 配合介接測試（6/9–6/30）

## 操作

單一 `index.html`，無相依套件。鍵盤 `←／→／空白鍵`翻頁、數字鍵跳頁、可觸控滑動；支援列印（每頁一張）。
