# 智慧閱讀平臺・資料介接協力說明

致 **線上資料庫得標廠商・資料庫維運廠商** 的資料介接配合事項簡報，供 **2026-06-08（民國115年）14:00 協調會** 使用。

🔗 **線上簡報**：https://norason666.github.io/reading-db-coop-1150608/

## 內容

從智慧閱讀系統的角度，說明本年度資料銜接作法與請廠商配合的六件事：

1. 書目欄位灌齊（syncDBBook 24 欄、17 必填）
2. 智慧閱讀視角的關鍵欄位務必提供（含與 Hamibook 欄位對照）
3. 書影規格（檔名＝書籍編號、≤60KB）
4. 書籍編號（bookID）一致性
5. 每日回傳瀏覽紀錄（getDBlog）
6. 配合介接測試（6/9–6/30）

並加入兩項請託：

- **給內容廠商**：把資料庫依主題／領域／學制切成多個子集合，各給一個客製化入口網址，由智慧閱讀平臺「線上資料庫」頁逐一呈現（今年得標 5 家 → 數十個主題入口，更豐富）。
- **給巨耀**：提供讓教育局／數位學習中心能自主、隨時、完整匯出全部資料庫內容（書目＋書影＋瀏覽紀錄）的功能，確保資料主導權與停服前承接。
